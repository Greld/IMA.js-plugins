## Installation

```javascript

npm install ima-plugin-analytic --save

```

```javascript
// /app/build.js

var vendors = {
	common: [
		'ima-plugin-analytic'
	]
};

/*
Now is analytic plugin available from:

ns.ima.plugin.analytic.Events
ns.ima.plugin.analytic.AbstractAnalytic

import { Events, AbstractAnalytic } from 'ima-plugin-analytic';
*/

```
